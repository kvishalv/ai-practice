# AI Practice Repo

This repo is for practicing AI pair-programming interviews.

## Goals

- Start from an empty or near-empty repo
- Practice planning before prompting AI
- Build incrementally in Python
- Keep code executable at every step
- Add tests early
- Extend architecture cleanly for follow-up requirements

## Recommended workflow

1. Open `INTERVIEW_PLAYBOOK.md`
2. Open `INTERVIEW_ASSISTANT_PROMPT.md`
3. Pick one problem from `problems/`
4. Start from the README only
5. Ask clarifying questions
6. Write requirements
7. Design the classes/files
8. Implement one component at a time
9. Run tests after each increment
10. Add one follow-up requirement

## Common problems

- LRU Cache
- Robot Grid Navigation
- Unix Tail

## Run tests

```bash
pytest
```

## Install dependencies

```bash
pip install -r requirements.txt
```
