# Problem: Robot Grid Navigation

Build an application to control robots navigating a grid with obstacles.

## Core requirements

- Represent a 2D grid
- Place obstacles
- Place a robot at a valid location
- Support commands like:
  - LEFT
  - RIGHT
  - MOVE
- Prevent invalid movement into walls or obstacles

## Suggested follow-ups

- multiple robots
- collision handling
- configurable map input
- path validation between points
- command scripts

## Design hint

Keep robot behavior separate from arena validation logic.
